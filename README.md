# Pl3BinaryReplacer

## 概要
**Pl3BinaryReplacer**は、指定されたフォルダ内の`.pl3`ファイルに含まれる特定の文字列を、ファイル名に基づいた識別子に置き換えるためのツールです。このプログラムは、Tkinterを使用して簡単なGUIを提供し、ファイルの一括置換を効率的に行います。

## 特徴
- `.pl3`拡張子を持つファイルに特化した置換処理
- バイナリファイルに対する安全なテキスト置換
- 置換結果のログを自動生成し、処理結果を記録
- GUIを介してユーザーフレンドリーな操作

## 使用方法
1. プログラムを起動します。
2. GUIが表示されたら、`フォルダを選択`ボタンをクリックして、処理したいフォルダを選択します。
3. フォルダが選択されると、そのフォルダ内のすべての`.pl3`ファイルに対して置換処理が行われます。
4. 処理が完了すると、置換結果のログファイルが生成され、処理の詳細が記録されます。

## 置換の仕様
- 対象ファイル: `x=`で始まり、`.pl3`で終わるファイル名を持つファイル
- 置換文字列: ファイル内の`"x=000"`という文字列を、ファイル名の`x=数字`に置換

## システム要件
- Python 3.x
- Tkinter（標準Pythonライブラリ）
- OS: Windows, macOS, Linux

## インストール方法
このプログラムはPythonスクリプトとして提供されています。以下の手順で使用できます。

1. [Python公式サイト](https://www.python.org/downloads/)からPython 3.xをインストールします（すでにインストールされている場合はこのステップをスキップしてください）。
2. プログラムのソースコードを任意のディレクトリに保存します。
3. コマンドラインまたはターミナルを開き、保存したディレクトリに移動します。
4. 以下のコマンドを実行してプログラムを起動します。

```bash
python Pl3BinaryReplacer.py
